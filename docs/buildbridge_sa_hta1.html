Hierarchical task analysis of purchasing surplus materials via BuildBridge SA, with five main subtasks and their decomposed steps.

0. Purchase surplus construction materials
via BuildBridge SA marketplace
Plan 0. Do 1 → 2 → 3 → 4 → 5 in sequence. If the desired product is not found at step 2, try 2.3 (change filter). If the cart is empty at step 4, return to step 2.
1. Land & orient
  • 1.1 Open site in browser
  • 1.2 View hero section (Builders / Contractors)
  • 1.3 Read impact stats
  • 1.4 Browse category tiles
  • 1.5 View featured listings
2. Search & filter
  • 2.1 Click "Shop" in nav
  • 2.2 Marketplace page loads
  • 2.3 Tick category filter
    • 2.3.1 Grid updates
    • 2.3.2 Result count shows
  • 2.4 Type search keyword
    • 2.4.1 Live filter on keystroke
    • 2.4.2 Non-matches hide
  • 2.5 Clear search field
3. Review & add to cart
  • 3.1 Identify product card
  • 3.2 Read name, location, price, badge
  • 3.3 (Opt) click heart → wishlist
    • 3.3.1 Heart fills orange
    • 3.3.2 Wishlist count updates
  • 3.4 Click "Add to cart"
    • 3.4.1 "✓ Added" shown 1.5s
    • 3.4.2 Cart icon increments
  • 3.5 Repeat for more products
4. Review cart & discount
  • 4.1 Open cart drawer
  • 4.2 Review items list
    • 4.2.1 Qty, price, remove shown
  • 4.3 Adjust quantity (±)
    • 4.3.1 Totals update live
  • 4.4 Review pricing breakdown
    • 4.4.1 Subtotal
    • 4.4.2 Bundle −10%
    • 4.4.3 Platform fee +7.5%
    • 4.4.4 Total in rands
  • 4.5 Click "Proceed to checkout →"
5. Complete checkout
  • 5.1 Checkout page loads
  • 5.2 Fill delivery details
    • 5.2.1 Full name
    • 5.2.2 Delivery address
    • 5.2.3 Contact number
  • 5.3 Review order summary
  • 5.4 Click "Complete order"
  • 5.5 Confetti + success message
  • 5.6 Order confirmed
