Hierarchical task analysis of purchasing materials for a 5-metre boundary wall using the BuildBridge SA Smart Estimator, with four main subtasks and their decomposed steps.

0. Purchase materials for a 5-metre boundary wall
using the BuildBridge SA Smart Estimator
Plan 0. Do 1 → 2 → 3 → 4 in sequence. If stock is unavailable at step 3.2, go to step 3.3 (browse shop manually). If checkout fails, retry step 4.
1. Access the Smart Project Estimator
  • 1.1 Open BuildBridge SA on mobile browser
  • 1.2 Observe landing page hero section
  • 1.3 Tap "Smart Estimator" in nav
    • Alt: tap CTA button on landing page
  • 1.4 Estimator page loads with two sliders (Wall Length, Wall Height)
2. Input project parameters
  • 2.1 Drag Wall Length slider to 5 m
    • 2.1.1 Live wall diagram updates proportionally
    • 2.1.2 Value below slider reads "5m"
  • 2.2 Drag Wall Height slider to 1.8 m
    • 2.2.1 Live wall diagram updates
    • 2.2.2 Value reads "1.8m"
3. Review output & select bundle
  • 3.1 Read result cards: Wall Area 5 m², Bricks 450, Cement 3 bags, 9 m³
  • 3.2 View Recommended Bundle panel
    • 3.2.1 Read bricks line item name & qty
    • 3.2.2 Read cement line item name & qty
    • 3.2.3 Read "After 10% bundle discount" price
  • 3.3 (If no stock) Read error → navigate to /shop manually
  • 3.4 Tap "Add Bundle to Cart (10% Off)"
    • 3.4.1 Button turns green "✓ Bundle Added to Cart!" 2s
    • 3.4.2 Cart icon updates with item count
4. Complete checkout
  • 4.1 Tap cart icon → open Cart Drawer
    • 4.1.1 Verify line items match bundle
    • 4.1.2 Verify 10% bundle discount deduction
    • 4.1.3 Verify 7.5% platform fee
    • 4.1.4 Verify final total in Rands
  • 4.2 Tap "Proceed to Checkout →"
  • 4.3 Complete checkout form (name, address, contact)
  • 4.4 Tap "Complete Order"
  • 4.5 Observe success screen with confetti
  • 4.6 Note order reference for WhatsApp follow-up
